🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Issue: Git push fails with authentication error
Fix: Follow Method A, B, or C in GITHUB-SETUP.md

Issue: GitHub Actions workflow fails
Fix: 
  1. Check Actions tab for detailed logs
  2. Verify all GitHub Secrets are configured correctly
  3. Test SSH connection manually: ssh -p PORT USERNAME@HOST

Issue: Application doesn't start on server
Fix:
  1. SSH into server
  2. Check logs: pm2 logs lifevision-healthcare
  3. Manual start: cd ~/public_html/current && node dist/server.cjs

Issue: Database not initializing
Fix:
  1. Ensure server/db.json exists
  2. Check write permissions: chmod -R 755 ~/public_html/current/server
